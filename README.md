# Kazmi_Creative Website Audit Tool (Flask Version)

This is a web-based version of the Kazmi_Creative website audit tool.

## Features
- **Page Speed Analysis** (Google PageSpeed API)
- **SEO Audit** (title, meta tags, headers, alt attributes)
- **Broken Link Detection**
- **Branded PDF Report** download

## How to Run
1. Install dependencies:
```bash
pip install -r requirements.txt
```

2. (Optional) Configure a Google PageSpeed API key to enable performance scoring:
```bash
export PAGESPEED_API_KEY="your_api_key_here"
```

3. Run the Flask app:
```bash
python app.py
```

4. Open the tool in your browser at:
```
http://127.0.0.1:5000
```
